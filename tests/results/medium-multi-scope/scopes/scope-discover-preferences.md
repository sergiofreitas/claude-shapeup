
# Scope: Discover Preferences From Dashboard

## Hill Position

▲ Uphill — trivially so. Only unknown is whether the dashboard view already has a guardian-role-gate helper or whether we're adding one. Five minutes of reading resolves it.

## Prioritization Reasoning

**Fourth scope. Last because it's lowest risk and the page it links to must exist first.** Building this any earlier means linking to a half-built page. Building it now means the feature is actually findable by users — without this, we've built a page nobody can reach.

Small scope, but essential: a feature that ships without a link into it is effectively not shipped.
